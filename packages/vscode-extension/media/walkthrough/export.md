## An editable .pptx, not a picture of one

`Lutrin: Export to PowerPoint (.pptx)` compiles the deck to a real
PowerPoint file:

- Text stays text, tables stay tables, shapes stay shapes — anyone can
  retouch the result without Lutrin.
- Reveal animations declared in Markdown become native PowerPoint
  animations.
- Mermaid diagrams are re-rendered as crisp high-resolution images,
  not screen captures.

The command is also in the editor title menu (`…`) and in the Explorer
right-click on any `*.deck.md`.
