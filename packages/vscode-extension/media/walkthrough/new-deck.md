## One heading per slide

A Lutrin deck is a Markdown file:

```markdown
---
title: My presentation
---

# First slide

- Write the content
- The engine lays it out

# Second slide

## Left column

## Right column
```

`Lutrin: New Presentation` opens a small deck that already compiles —
edit it and watch the preview follow.

Files named `*.deck.md` (or carrying `deck: true` in their frontmatter)
are validated automatically, even with no preview open.
